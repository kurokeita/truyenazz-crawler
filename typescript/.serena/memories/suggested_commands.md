# Suggested Commands
- Install dependencies: `npm install`
- Build package: `npm run build`
- Run CLI directly: `node dist/cli.js "https://truyenazz.me/your-novel"`
- Run interactive mode: `node dist/cli.js --interactive`
- Start built package via script: `npm start`
- Published-style usage after packaging/publishing: `pnpx @kurokeita/truyenazz-crawler "https://truyenazz.me/your-novel"`
- Inspect repository files on Darwin: `ls`, `find`, `rg`, `git status`, `git diff`
